── GitHub Integration (added 2026-05-24) ────────────────────────────────────
The project is managed in GitHub under: https://github.com/stevebuonincontri/thumbnail-creator

GitHub credentials are stored in .env (never committed):
  GITHUB_TOKEN    — personal access token with repo scope
  GITHUB_USERNAME — stevebuonincontri
  GITHUB_REPO     — thumbnail-creator
  GITHUB_BRANCH   — main

src/github_manager.py handles all git/GitHub operations:
  python3 src/github_manager.py           # init local repo + create remote + push
  python3 src/github_manager.py --push    # push only
  python3 src/github_manager.py --status  # show git status

The thumbnail generator also accepts --push to sync after generation:
  python3 src/thumbnail.py --push

Safe-to-commit template for all env vars: .env.example
Output images (output/*.png) are gitignored.
